## üõ†Ô∏è The Philosophy
StoryCraft uses a modular approach. The Core Engine handles the "How" of the game, while Settings handle the "What" and "Where".

*   **Universality**: A character from a Fantasy setting should be mechanically compatible with a Sci-Fi setting.
*   **Narrative First**: Mechanics are designed to support and drive the story, not just simulate physics.
*   **Simplicity**: Easy to learn, but with depth provided by modular expansions.
